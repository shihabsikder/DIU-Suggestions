## Step 2: Extract the ZIP File

### **On Windows:**
1. Find the downloaded file: `diu-suggestions-main.zip`
2. **Right-click** on it
3. Click **"Extract All"**
4. Choose a folder (e.g., Desktop or Documents)
5. Click **"Extract"**

## ✅ Troubleshooting

### **Website won't load?**

**Solution 1:** Make sure you have Chrome installed
- Download from: https://www.google.com/chrome/

**Solution 2:** Try dragging the file again
- Some browsers need a fresh load

**Solution 3:** Check file path
- Make sure `index.html` is in the main folder (not in a subfolder)

**Solution 4:** Clear Chrome Cache
- Press: `Ctrl + Shift + Delete` (Windows) or `Cmd + Shift + Delete` (Mac)
- Click "Clear Data"
